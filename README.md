## Overview

This project is an automation testing framework developed for the **Demo Web Shop** application using **Java, Selenium WebDriver, TestNG, Maven**, and the **Page Object Model (POM)** design pattern.

It was developed as part of the **Digital Egypt Pioneers Initiative (DEPI) Software Testing Graduation Project**.

The framework automates end-to-end user workflows while following automation testing best practices to ensure maintainable, reusable, and scalable test automation.

**Website Under Test:**  
https://demowebshop.tricentis.com/
